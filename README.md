## 🚀 Running Locally

```bash
# Clone the repository
git clone https://github.com/Princ3mish/Portfolio.git
cd Portfolio

# Install dependencies
pnpm install

# Start development server (webpack mode)
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
